## Basic Examples

## Running the Examples

All examples are standalone Python scripts that you can run directly:

```bash
# Clone the repository
git clone https://github.com/fpetranzan/mimitFuelPy.git
cd mimitFuelPy

# Install the library
pip install -e .

# Run examples
python examples/basic_usage.py
```
